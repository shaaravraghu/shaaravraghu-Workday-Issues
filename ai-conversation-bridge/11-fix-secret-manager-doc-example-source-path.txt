Suggested Title: Fix the Secret Manager deployment example to use `--source chat-connector` instead of `--source .`

## Description
The setup guide’s Secret Manager example switches from `--source chat-connector` to `--source .`. From the repository root, `--source .` points Cloud Run at the whole repository rather than the `chat-connector` service directory. That is inconsistent with the earlier deploy command and is likely to break the example.

## Actual Behavior
